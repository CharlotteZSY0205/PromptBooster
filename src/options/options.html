How to use PromptBooster

  1. Paste your API key above. PromptBooster never sends traffic without it.
  2. Open ChatGPT and look for the new Boost Prompt button next to Send.
  3. Select preview mode if you want to review the rewrite before it is auto-submitted.

Reminder: your prompts are sent directly from the browser to the API provider you configure here. PromptBooster does not operate any servers or store your data.
